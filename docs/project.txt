# Folder Responsibilities (LOCKED)

### `auth/`

Reusable authentication.

Endpoints:

```plaintext
POST /auth/register
POST /auth/login
POST /auth/logout
POST /auth/refresh
POST /auth/forgot-password
POST /auth/reset-password
POST /auth/verify-email
```

---

### `users/`

User account management.

Endpoints:

```plaintext
GET /users/me
PATCH /users/me
DELETE /users/me
```

Optional later:

```plaintext
GET /users
GET /users/{id}
```

---

### `database/`

Only:

* PostgreSQL connection
* SQLAlchemy setup
* Alembic migrations

No business logic.

---

### `core/`

Shared infrastructure:

* Config
* JWT
* Password hashing
* Responses
* Exceptions
* Shared dependencies

---

### `utils/`

Pure reusable functions.

Examples:

```plaintext
send_email()
validate_email()
normalize_string()
```

No business rules.

---

### `tests/`

Tests only.

---
